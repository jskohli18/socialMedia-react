# Social Media Dashboard

A simple, responsive single-page React application that simulates a live social media feed. 

lIve Website Link: `https://socialmediajsk.netlify.app/`

# Features
--View Posts: Automatically fetches and displays a feed of posts when the app loads. <br/>
--Create Posts: Fill out a form to instantly add new posts with custom titles, tags, and reactions. <br/>
--Delete Posts: Click the trash icon to remove posts from the feed immediately. <br/>
--Smooth Navigation: Switch between the home feed and the creation form without the browser reloading. <br/>

# Tech Stack 
--Framework: React.js  <br/>
--State Management: Context API & useReducer  <br/>
--Routing: React Router   <br/>
--Styling: Bootstrap 5 & React Icons  <br/>
--Data: DummyJSON API
